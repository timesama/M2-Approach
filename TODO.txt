TASK 6 — Processing pipeline

Preserve the old mathematical workflow but reorganize it.

Controller methods may be:

    reset_tab()
    connect_signals()
    load_fid_files()
    load_data_files()
    load_empty_fid_files()
    load_empty_data_files()

    build_up()
    run_time_analysis()
    save_statistics()
    export_se_max_t2()
    open_manual_echo_time_dialog()

Numerical functions must be in calculations/recfid_signal.py.

The old calculation flow to preserve:

1. Read FID and data files.
2. Apply time-domain correction:
    - phase
    - optional smoothing
    - optional empty subtraction
    - optional zero shift
    - optional cut beginning
    - optional normalize-to-FID
    - optional long-component subtraction
3. Optional time-domain apodization.
4. Frequency-domain correction:
    - FFT
    - optional phase/frequency adjustment
    - zero filling
    - baseline correction
    - FFT apodization
5. For SE mode:
    - process all SE files
    - infer echo time from filename if possible
    - if echo time cannot be read, open manual echo-time assignment dialog
    - calculate max amplitude for each SE
    - extrapolate maximum SE amplitude
6. For MSE mode:
    - process the single data file
    - maximum/extrapolation comes from max processed data signal
    - divide by user-controlled MSE divider
7. Build up reconstructed FID from original FID and extrapolation.
8. Plot original and built-up time/frequency signals.
9. Save/export RecFID-specific results using the old software’s save behavior.
